# 静态分支说明

本分支为纯静态分支，包含项目的静态资源文件。可以将本分支部署到任意支持静态文件托管的平台，例如：

- [GitHub Pages](https://pages.github.com/)
- [Vercel](https://vercel.com/)
- [Netlify](https://www.netlify.com/)

## 部署步骤

1. 选择一个静态托管平台。
2. 将本分支的代码上传到平台。
3. 部署完成后，平台会生成一个访问地址。
4. 使用生成的地址将内容展示到前端站点。
